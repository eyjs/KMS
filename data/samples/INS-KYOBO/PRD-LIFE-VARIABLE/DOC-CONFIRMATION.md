## 확인 내용

### 1. 상품 설명 확인

본인은 교보생명 변액보험에 대하여 
다음 사항을 충분히 설명받았음을 확인합니다.

- [ ] 보장 내용 및 보험금 지급사유
- [ ] 보험금을 지급하지 않는 사유
- [ ] 면책기간 및 감액기간
- [ ] 보험료 납입기간 및 납입방법
- [ ] 해약환급금
- [ ] 청약철회 및 품질보증해지 권리

### 2. 비교 설명 확인 (500인 이상 GA)

본인은 3개 이상 유사 상품에 대한 비교 설명을 받았습니다.

| 비교 항목 | 교보생명 | 비교사1 | 비교사2 |
|----------|----------------|---------|---------|
| 상품명 | 변액보험 | | |
| 보험료 | | | |
| 주요보장 | | | |

### 3. 적합성 확인

설계사가 본인의 다음 정보를 파악하였습니다.

- [ ] 재산상황
- [ ] 금융상품 가입 경험
- [ ] 금융상품에 대한 이해도
- [ ] 가입 목적

### 4. 기타 확인

- [ ] 기존 보험계약 해지에 따른 불이익을 설명받았습니다.
- [ ] 중요사항은 설계사가 직접 설명하였습니다.
- [ ] 전자서명의 법적 효력에 대해 이해하였습니다.
