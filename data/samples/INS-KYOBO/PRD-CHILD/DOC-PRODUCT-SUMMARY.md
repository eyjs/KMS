# 교보생명 어린이보험 상품요약서

---

## 1. 상품 특징

| 구분 | 내용 |
|------|------|
| 상품유형 | 보장성보험 |
| 보험기간 | 종신 / 80세만기 / 100세만기 |
| 납입기간 | 10/15/20/30년납 |
| 가입연령 | 만 15세 ~ 만 70세 |

---

## 2. 주요 보장 내용

### 기본계약
- 사망보험금: 가입금액 100%
- 재해사망추가: 가입금액 100%

### 필수 확인사항
- 보험금 지급제한 사유 확인
- 면책기간 및 감액기간 확인
- 갱신형 특약의 보험료 변동 가능성

---

## 3. 해약환급금

## 4. 기타 유의사항

1. 청약철회: 보험증권 수령일로부터 15일 이내
2. 품질보증해지: 3개월 이내 계약내용과 다를 경우
3. 위법계약해지: 5년 이내 설명의무 위반 시
