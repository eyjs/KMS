```
가망고객 발굴 → 상담 → 설계 → 청약 → 심사 → 증권발급
```

| 단계 | 소요시간 | 담당 |
|------|----------|------|
| 상담 | 30분~1시간 | 설계사 |
| 설계 | 10분 | 설계사 |
| 청약 | 20분 | 설계사+고객 |
| 심사 | 1~5일 | 심사팀 |
