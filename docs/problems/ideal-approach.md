# 2. 이상적 기술 스택

## 2.1 아키텍처

```
┌─────────────────────────────────────────────────────────┐
│                    Vue 3 Frontend                        │
│  • Admin Dashboard                                       │
│  • 문서 뷰어                                             │
│  • 지식 그래프 시각화                                    │
└──────────────────────┬──────────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────────┐
│                 .NET Core API Gateway                    │
│  • 인증/권한 (JWT)                                       │
│  • 라우팅                                                │
│  • 비즈니스 로직                                         │
└──────────────────────┬──────────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────────┐
│               Python Core Engine                         │
│  • PDF 텍스트 추출 (pdfplumber)                          │
│  • 한글 NLP (Kiwi/KoNLPy)                                │
│  • 자동 분류 (ML 기반)                                   │
│  • 관계 추론 (Rule + ML)                                 │
│  • 임베딩 생성 (sentence-transformers)                   │
└──────────────────────┬──────────────────────────────────┘
                       │
       ┌───────────────┼───────────────┐
       ▼               ▼               ▼
┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ PostgreSQL  │ │ Neo4j       │ │ Vector DB   │
│ 메타데이터   │ │ 지식그래프   │ │ 임베딩      │
└─────────────┘ └─────────────┘ └─────────────┘
                       │
┌──────────────────────▼──────────────────────────────────┐
│                   RAG Pipeline                           │
│  • LangChain / LlamaIndex                                │
│  • LLM 연동 (GPT-4 / Claude)                             │
│  • 챗봇 인터페이스                                       │
└─────────────────────────────────────────────────────────┘
```

## 2.2 기술 스택 상세

| 레이어 | 기술 | 역할 |
|--------|------|------|
| Frontend | Vue 3 + Vite | 사용자 인터페이스 |
| API Gateway | ASP.NET Core 8 | 인증, 라우팅, 비즈니스 로직 |
| Core Engine | Python 3.11 | 문서 처리, NLP, ML |
| RDBMS | PostgreSQL 16 | 메타데이터, 분류, 관계 |
| Graph DB | Neo4j 5.x | 지식 그래프, 복잡한 관계 탐색 |
| Vector DB | pgvector / Pinecone | 임베딩 저장, 유사도 검색 |
| RAG | LangChain | 문서 기반 AI 응답 |
| LLM | GPT-4 / Claude | 자연어 이해, 생성 |

# 4. 이상적 기능

## 4.1 문서 처리

| 기능 | 설명 |
|------|------|
| PDF 텍스트 추출 | 표, 이미지 캡션 포함 95%+ 정확도 |
| OCR | 스캔 문서 처리 |

| 언어 감지 | 한글/영문 자동 판별 |

## 4.2 자동 분류
