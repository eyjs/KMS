# 삼성생명 심사 기준

## 질병별 심사 기준

### 고혈압
| 혈압 수치 | 심사 결과 |
|----------|----------|
| 140/90 미만 | 표준체 |
| 140-160/90-100 | 부표준체 (할증 10%) |
| 160 이상 | 거절 또는 부담보 |

### 당뇨
| HbA1c | 심사 결과 |
|-------|----------|
| 6.5 미만 | 표준체 |
| 6.5-7.5 | 부표준체 |
| 7.5 이상 | 거절 |

## 직업별 심사 기준
- 1급(사무직): 표준
- 2급(경작업): 표준
- 3급(중작업): 할증 또는 제한
- 4급(위험직): 거절
