## 환수 면제 사유

### 면제 대상
1. 피보험자 사망
2. 보험사 귀책사유 해지
3. 청약철회 (정상 절차)
4. 천재지변

### 부분 면제
- 품질보증해지: 50% 감면
- 무효: 사유에 따라 판단

```
해지 발생 → 환수 대상 확인 → 환수액 계산 → 다음 정산에서 차감
```

## 환수 방지 방법

### 해지 예방
1. 완전판매 철저
2. 고객 상황 파악
