## 2. 주요 기능

### 청약
1. 고객 정보 입력
2. 상품 선택
3. 설계 저장
4. 전자청약 진행
5. 서류 업로드

### 실적 조회
- 일별/월별 실적
- 시책 달성률
- 수수료 예상액

### 고객 관리
- 고객 DB 조회
- 계약 현황
- 만기 알림
- 상담 이력

## 3. 전자청약 프로세스

```
설계 생성 → 고객 정보 입력 → 보장 설계
    ↓
전자서명 요청 → 고객 서명 → 청약 완료
    ↓
심사 → 승낙 → 증권 발급
```

### 주의사항
- 서명 전 중요사항 설명
- 서명 완료 후 수정 불가
- 24시간 내 서명 필요

## 4. 문제 해결

### 로그인 안 됨
1. 비밀번호 확인 (초기: 생년월일)
2. 계정 잠금 시 고객센터 연락

### 전자서명 오류
1. 앱 업데이트 확인
2. 네트워크 연결 확인
3. 고객 본인인증 재시도
