---

## 학습 목표

1. 변액보험의 상품 구조를 이해한다
2. 경쟁사 대비 차별점을 파악한다
3. 효과적인 판매 전략을 습득한다

## 1. 상품 개요

### 상품 특징
| 항목 | 내용 |
|------|------|
| 상품명 | 교보생명 변액보험 |
| 보험종목 | 생명보험 |
| 보험기간 | 종신/80세/100세 |
| 납입기간 | 10/15/20/30년납 |

### 타깃 고객
- 가장의 책임감이 있는 30~50대
- 종합적인 보장을 원하는 고객
- 저축과 보장을 동시에 원하는 고객

## 4. 판매 포인트

### Key Message
> "한 번 가입으로 평생 보장, 합리적인 보험료"

### 3가지 핵심 포인트
1. **평생 보장**: 갱신 없는 비갱신형
2. **폭넓은 보장**: 3대 질병 + 사망
3. **합리적 보험료**: 업계 최저 수준

## 5. FAQ

**Q1: 보험료가 비싸지 않나요?**
→ A: 납입기간을 길게 선택하면 월 보험료를 낮출 수 있습니다.

**Q2: 나중에 보험료가 오르나요?**
→ A: 비갱신형이라 처음 가입한 보험료가 끝까지 유지됩니다.

**Q3: 다른 보험과 중복 가입 가능한가요?**
→ A: 네, 실손의료보험 외에는 중복 가입 가능합니다.
