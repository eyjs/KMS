# 컴플라이언스 교육

---

## 1. 컴플라이언스란?

법규 준수 + 윤리 준수 + 내부통제

### 왜 중요한가?
- 고객 보호
- 회사 신뢰도
- 설계사 보호

---

## 2. 완전판매 의무

### 설명의무 (금소법 제19조)

**반드시 설명할 사항**
1. 보험금 지급사유
2. 보험금을 지급하지 않는 사유
3. 면책기간/감액기간
4. 해약환급금
5. 청약철회권

**설명방법**
- 고객이 이해할 수 있는 수준으로
- 중요사항은 명확하게
- 설명 후 서명 징구

### 적합성 원칙 (금소법 제17조)

**파악 정보**
- 재산상황
- 금융상품 경험
- 이해도
- 연령
- 가입 목적

---

## 3. 불완전판매 유형

| 유형 | 설명 |
|------|------|
| 설명의무 위반 | 중요사항 미설명 |
| 자필서명 미이행 | 대리서명, 전자서명 오류 |
| 서류 미교부 | 약관, 청약서 부본 미전달 |
| 부당승환 | 기존계약 불이익 미설명 |
| 부실고지 권유 | 고지사항 축소 권유 |
| 허위설명 | 보장내용 과장/왜곡 |

---

## 4. 제재

### 설계사 제재
| 위반 | 제재 |
|------|------|
| 경미한 위반 | 경고, 주의 |
| 중대한 위반 | 업무정지 (1~6개월) |
| 반복 위반 | 등록취소 |

### 형사처벌
- 3년 이하 징역
- 3천만원 이하 벌금

---

## 5. 해피콜 대응

### 해피콜이란?
- 청약 후 완전판매 여부 확인 전화
- 청약철회 기간 내 실시

### 확인 내용
1. 설계사로부터 설명 받았는지
2. 서류를 받았는지
3. 자필서명 했는지
4. 청약철회 안내 받았는지

### 대응 요령
- 정확하게 답변
- 모르면 "모른다" 답변
- 거짓 답변 금지

---

## 6. 체크리스트

### 청약 전
□ 적합성 확인 완료
□ 중요사항 설명 완료
□ 비교설명 완료 (대리점)

### 청약 시
□ 자필서명 확인
□ 서류 교부 완료
□ 확인서 징구

### 청약 후
□ 해피콜 안내
□ 증권 전달 확인
