# 삼성생명 어린이보험 상품설명서

## 상품 개요
삼성생명의 대표 어린이보험입니다.

## 주요 보장 내용
- 사망보험금: 가입금액의 100%
- 재해사망보험금: 가입금액의 100% 추가

## 가입 조건
- 가입연령: 15세 ~ 70세
- 납입기간: 10년, 15년, 20년
- 보험기간: 종신
